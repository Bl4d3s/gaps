Gaps - Find your missing movies

Welcome to Gaps

Gaps searches through your Plex Server. It then queries for known movies in the same collection using The MovieDB, which you will need to get. Don't worry, we'll walk you through those steps. Anyway, if those movies don't exist in your library, Gaps will recommend getting those movies, legally of course.

An example would be owning 'Alien (1979)' and Gaps recommending 'Aliens (1986)' and 'Alien³ (1992)' to be added to your collection.

To use Gaps, you'll need a MovieDB api key. Navigate over to The Movie DB, create an account, and make an API Key. Copy that key and paste it below.
